# 🚀 CodeAlpha Internship Tasks

Welcome to my **CodeAlpha Internship Repository**!

This repository contains the tasks and projects I completed during my internship at **CodeAlpha**. The objective of this internship was to gain practical experience in the Internet of Things (IoT), strengthen problem-solving skills, and understand how connected systems are designed and implemented.

Each task has helped me improve my understanding of IoT concepts, embedded systems, documentation, and project development.

## 📂 Repository Contents

### 📄 Task 1 – Research Report: IoT in Real Life (Smart Home)

A research report discussing how the **Internet of Things (IoT)** is transforming modern homes into smart, connected environments. The report covers IoT concepts, smart home applications, benefits, challenges, future scope, and references.

**Topics Covered**

* Introduction to IoT
* Smart Home Technology
* Smart Lighting
* Smart Security Systems
* Smart Appliances
* Smart Thermostats
* Voice-Controlled Assistants
* Benefits & Challenges
* Future of Smart Homes
* References

### 🚗 Task 2 – Smart Parking System (Sensor-Based Simulation)

A simulated IoT-based Smart Parking System developed using **ESP32** and **Wokwi**.

#### Features

* Detects vehicle availability using sensors
* Displays parking slot status
* Entry and exit monitoring
* Real-time simulation
* Practical IoT implementation

#### Technologies Used

* ESP32
* Wokwi Simulator
* PlatformIO
* C++
* Virtual Sensors
